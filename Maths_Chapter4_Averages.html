🌟 What is an Average?

An average is a single number that represents a whole set of data. If lots of people rate a film, the average score tells you what most people thought about it. You can use averages to compare, summarise and make decisions!

🎬 Example: A film gets these scores from 9 people:
6 · 8 · 6 · 5 · 7 · 6 · 4 · 9 · 6
The average score helps you decide whether to watch it!

🔑 Key Words

1
Average — a single value that represents a whole data set. It tells you what is typical for that data.
2
Mode — the value that appears most often in a data set. It is the most popular value. There can be more than one mode, or no mode!
3
Median — the middle value when all values are sorted in order from smallest to largest. Half the values are below it, half above.

🎯 What is the Mode?

Mode = the value that appears the MOST often in a data set.
Think of it as the most popular value — just like the most popular flavour of ice cream!

How to find the Mode

1
Look at all the values in the data set.
2
Count how many times each value appears.
3
The value with the highest count is the mode!

📦 Tip to remember

If you have: cat, cat, mouse, rabbit — "cat" appears the most, so the mode is cat! The mode works for words too, not just numbers.

📝 Worked Examples

Example 1 — Numbers
Data: 8, 9, 9, 8, 9, 8, 9
Count 8s: 3 times
Count 9s: 4 times
∴ Mode = 9 (appears 4 times)
Example 2 — Two modes!
Data: 11, 11, 11, 13, 13, 13
Count 11s: 3 times
Count 13s: 3 times
∴ There are two modes: 11 and 13
Example 3 — Bananas in bunches
Data: 6, 4, 5, 6, 7, 4, 4, 5, 6, 5, 7, 5
Count 4s: 3 · Count 5s: 4 times · Count 6s: 3 · Count 7s: 2
∴ Mode = 5 bananas per bunch

🏪 Gloves Problem

A shop sells gloves in sizes 1, 2, 3 and 4. In one day, the shop sells these sizes:

1 · 2 · 3 · 4 · 1 · 4 · 4 · 2 · 4
Mode = 4 (appears 4 times!)
The shopkeeper should order more size 4 gloves — they are the most popular!

Count: Size 1 → 2× Size 2 → 2× Size 3 → 1× Size 4 → 4×

💡 Why Mode is useful here

The mode tells you the most popular size. If the shopkeeper wants to sell more gloves, they should stock up on the most popular size — that's the mode!

📏 What is the Median?

Median = the MIDDLE value when all values are sorted from smallest to largest.
Half the values are below the median, and half are above it!

The Crossing-Out Method

1
Write all the values in order from smallest to largest.
2
Cross out the first and last values at the same time.
3
Keep crossing out from both ends together until one value remains.
4
The value left in the middle is the median!

📝 Worked Example

Finding the median of: 133 cm, 137 cm, 134 cm, 131 cm, 132 cm
Step 1 — Sort:
131, 132, 133, 134, 137

Step 2 — Cross out first pair:
131, 132, 133, 134, 137

Step 3 — Cross out next pair:
131, 132, 133, 134, 137

Answer: Median = 133 cm

💡 What if there's an even number of values?

With an odd number of values, there is always one middle value — that is the median.

In Cambridge Stage 5, you will always be given an odd number of values to work with, so there will always be a single middle value!

🤔 Think about it!

If you have 5 values, after sorting, the 3rd value is in the middle. If you have 7 values, the 4th value is in the middle. If you have 9 values, the 5th value is in the middle. Can you see the pattern?

⚖️ Mode vs Median — Which is better?

Both mode and median are averages, but sometimes one is more useful than the other! The key is to think about what the question is asking.

🎯 Use the MODE when...
• You want to know the most popular or common value
• You are choosing what to stock in a shop
• You want to find the most typical item
📏 Use the MEDIAN when...
• You want the middle of a range
• There are very high or low values that might skew the data
• You want a fair middle point

Find the mode of each set of numbers.

Working space...

a) 1 appears 3 times, 2 once, 3 once → Mode = 1

Question 3 [6 marks]

Find both the mode and the median of each set. Write your answers in the sentence.

a)   3, 5, 6, 2, 9, 3, 4, 3, 5

"The mode is _______ and the median is _______"    [3]

b)   5, 2, 5, 6, 7, 2, 3, 2, 1

"The mode is _______ and the median is _______"    [3]

Working space — sort the data first!
✅ Worked Solution:
a) Data: 3, 5, 6, 2, 9, 3, 4, 3, 5
Count: 2→1, 3→3, 4→1, 5→2, 6→1, 9→1 → Mode = 3
Sorted: 2, 3, 3, 3, 4, 5, 5, 6, 9 → middle (5th) = 4 → Median = 4
Answer: "The mode is 3 and the median is 4"

b) Data: 5, 2, 5, 6, 7, 2, 3, 2, 1
Count: 1→1, 2→3, 3→1, 5→2, 6→1, 7→1 → Mode = 2
Sorted: 1, 2, 2, 2, 3, 5, 5, 6, 7 → middle (5th) = 3 → Median = 3
Answer: "The mode is 2 and the median is 3"
Question 4 [5 marks]

Lucas has been asked to report the average amount of money raised at an event. The amounts raised were:

$9   $7   $6   $5   $9

a)   What is the mode of the amounts?     Mode = _________       [1]

b)   What is the median amount?     Median = _________       [2]

c)   Is it better to use the mode or the median to describe the average amount of money raised? Give a reason.       [2]

Write your answer for part c) here...
✅ Worked Solution:
a) $9 appears twice (most) → Mode = $9

b) Sorted: $5, $6, $7, $9, $9 → middle (3rd) = $7 → Median = $7

c) The median ($7) is better to use here.
Reason: The mode ($9) only counts the most common amount — but only 2 out of 5 people gave $9. The median ($7) gives a fairer picture of what a typical person donated, as it is the middle value of the sorted list.

b)   Find the median of the data.     Median = _________       [2]

c)   Is it better to use the mode or the median to decide what size tables will be most useful? Give a reason.       [1]

Working space for parts a), b), c)...
✅ Worked Solution:
